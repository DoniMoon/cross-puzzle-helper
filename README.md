# Cross-puzzle-helper

![fig](/description.png)

## Start
```bash
conda create --n your_env python=3.11.5
conda activate your_env
pip install -r requirements.txt
voila solver.ipynb
```

## How to use
1. Type setting
2. Get candidate
3. Get feedback to model by checking the checkbox next to the suggestion
